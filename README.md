**Mailer** is a lightweight command-line email utility built for developers and learners.
Designed with a clean terminal UI using `rich` — runs smoothly on Termux and Linux.

> ⚠️ **Educational Purposes Only** — This project is built to demonstrate how email automation works using Python. Any misuse for spam, phishing, or unauthorized activity is strictly prohibited. The author is not responsible for any misuse.

---

## ✨ Features

- 🎨 **Clean Terminal UI** — Minimal and elegant interface using `rich`.
- ⚡ **Super Lightweight** — No unnecessary dependencies.
- 📦 **Quick Setup** — Clone, install, and you're good to go.
- 🔐 **Transparent Codebase** — Simple code, easy to understand and learn from.

---

## ⚙️ Installation

### 📱 Termux

```bash
pkg update && pkg upgrade
pkg install python git
git clone https://github.com/akazatec/mailer
cd mailer
bash setup.sh
```

### 🐧 Linux

```bash
sudo apt update && sudo apt upgrade
sudo apt install python3 python3-pip git
git clone https://github.com/akazatec/mailer
cd mailer
bash setup.sh
```

---

## 🚀 Usage

```bash
python3 mailer.py
```

On-screen prompts follow karein — email, subject aur message enter karo.
